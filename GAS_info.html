1. 変数・定数

変数と定数はデータを保存するための箱のようなものです。JavaScriptでは、letとconstを使って変数や定数を宣言します。変数はプログラムの実行中に値を変更することができますが、定数は一度設定した値を変更することはできません。

  • let: 再代入可能な変数。例: let num = 3;。この変数は後から値を変更できます。例えば、num = 5;とすることで、変数numの値を3から5に変更することができます。
  • const: 再代入不可能な定数。例: const pi = 3.14;。この定数は一度値を設定すると変更できません。例えば、円の計算に使う円周率など、変更する必要のない値を定数として宣言します。
  • =: 代入演算子。右側の値を左側の変数に格納します。例: let greeting = "こんにちは";。この場合、文字列"こんにちは"が変数greetingに格納されます。

変数や定数を使うことで、プログラム内でデータを管理しやすくなります。例えば、ユーザーの入力を保存したり、計算結果を保持したりするのに役立ちます。変数や定数を適切に使うことで、プログラムの可読性や保守性が向上します。

練習問題

変数と定数を使って、以下の操作を行ってください。

  1. 変数numを宣言し、値を10に設定してください。
  2. 定数greetingを宣言し、値を"こんにちは"に設定してください。
  3. 変数numの値を20に変更し、コンソールに出力してください。

変数の宣言：
let 変数名 = 値;

定数の宣言：
const 定数名 = 値;


let num = 10;
const greeting = "こんにちは";
num = 20;
console.log(num);  // 20を出力

2. 順次処理

プログラムは上から下へ順番に実行されます。これを順次処理と呼びます。順次処理では、コードが書かれた順番に従って実行されるため、プログラムの流れを理解しやすくなります。console.log()を使って、データをコンソールに出力できます。これはデバッグやプログラムの動作確認に非常に便利です。


let greeting = "こんにちは";
console.log(greeting);  // "こんにちは"を出力

このように、console.log()を使うこで、変数の値や計算結果を確認することができます。例えば、複雑な計算を行う前に、変数の値を確認しておくことで、プログラムの誤りを早期に発見することができます。

また、順次処理を理解することで、プログラムの流れを予測しやすくなり、バグの原因を特定しやすくなります。順次処理はプログラムの基本的な実行順序であり、他の制御構造（分岐処理や繰り返し処理）と組み合わせて使うことで、より複雑な処理を実現することができます。

練習問題

順次処理を使って、以下の操作を行ってください。

  1. 変数nameを宣言し、値を"太郎"に設定してください。
  2. 変数ageを宣言し、値を25に設定してください。
  3. 変数nameとageの値をコンソールに出力してください。

変数の宣言：
let 変数名 = 値;

コンソール出力：
console.log(変数名);


let name = "太郎";
let age = 25;
console.log(name);  // "太郎"を出力
console.log(age);   // 25を出力

3. 分岐処理

条件に応じて異なる処理を行います。分岐処理を使うことで、プログラムの実行フローを制御することができます。if文を使って条件を設定し、条件が真の場合に特定の処理を実行します。例えば、ユーザーの入力に応じて異なるメッセージを表示する場合などに使います。

  • ===: 等価演算子。例: if (num === 3)。左右の値が同じかどうかを比較します。例えば、numが3であれば条件は真となり、if文の中の処理が実行されます。
  • !==: 不等価演算子。例: if (num !== 3)。左右の値が異なるかどうかを比較します。例えば、numが3でなければ条件は真となり、if文の中の処理が実行されます。
  • >, <: 大小比較演算子。例: if (num > 0)。数値の大小を比較します。例えば、numが0より大きければ条件は真となり、if文の中の処理が実行されます。

let num = 5;
if (num > 3) {
    console.log("3より大きい");
} else {
    console.log("3以下");
}

このように、条件に応じて異なるメッセージを出力することができます。分岐処理を使うことで、プログラムの柔軟性が向上し、さまざまな状況に対応することができます。

また、else ifを使うことで、複数の条件を設定することができます。例えば、数値が正の数か負の数か、またはゼロかを判定する場合などに使います。

練習問題

分岐処理を使って、以下の操作を行ってください。

  1. 変数scoreを宣言し、値を85に設定してください。
  2. 変数scoreの値が90以上の場合は"優秀"、70以上90未満の場合は"良好"、それ以外の場合は"努力が必要"とコンソールに出力してください。

条件分岐の書き方：
if (条件) {
処理
} else if (条件) {
処理
} else {
処理
}


let score = 85;
if (score >= 90) {
    console.log("優秀");
} else if (score >= 70) {
    console.log("良好");
} else {
    console.log("努力が必要");
}

4. 繰り返し処理

同じ処理を複数回実行する場合に使用します。繰り返し処理を使うことで、同じコードを何度も書く必がなくなり、プログラムが簡潔になります。主にfor文とwhile文があります。


// for文の基本形
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0から4まで出力
}

// while文の基本形
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

for文は、指定した回数だけ繰り返し処理��行います。例えば、for (let i = 0; i < 5; i++)とすることで、iが0から4までの間、繰り返し処理が実行されます。

while文は、条件が真である限り繰り返し処理を行います。例えば、while (count < 3)とすることで、countが3未満である間、繰り返し処理が実行されます。

繰り返し処理を使うことで、同じ処理を効率的に行うことができます。例えば、配列の要素を順番に処理する場合や、特定の条件が満たされるまで処理を繰り返す場合などに使います。

練習問題

繰り返し処理を使って、以下の操作を行ってください。

  1. 1から10までの数字を順番に出力してください。
  2. 変数sumを宣言し、1から10までの数字の合計を計算してコンソールに出力してください。

for文の書き方：
for (let i = 開始値; i <= 終了値; i++) {
処理
}


for (let i = 1; i <= 10; i++) {
    console.log(i);
}

let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum += i;
}
console.log(sum);  // 55を出力

5. 関数

関数は、特定の処理をまとめて名前を付けたものです。関数を使うことで、コードの再利用性が高まり、プログラムが整理されます。関数は、必要なときに呼び出して実行することができます。


function greet(name) {
    console.log("こんにちは、" + name + "さん");
}

greet("太郎");  // "こんにちは、太郎さん"を出力

このように、関数を定義して呼び出すことで、同じ処理を何度も実行することができます。関数は、引数を受け取ることができ、引数に基づいて異なる処理を行うことができます。

例えば、greet関数は、引数nameを受け取り、"こんにちは、" + name + "さん"というメッセージをコンソールに出力します。関数を使うことで、コードの重複を避け、プログラムの可読性や保守性を向上させることができます。

また、関数は戻り値を持つことができます。例えば、計算結果を返す関数や、特定の条件に基づいて値を返す関数などがあります。

練習問題

関数を使って、以下の操作を行ってください。

  1. 引数として2つの数値を受け取り、その合計を返す関数addを定義してください。
  2. 関数addを呼び出し、結果をコンソールに出力してください。

関数の定義：
function 関数名(引数1, 引数2) {
処理
return 戻り値;
}


function add(a, b) {
    return a + b;
}

let result = add(5, 3);
console.log(result);  // 8を出力

6. オブジェクト

オブジェクトは、関連するデータや機能をまとめたものです。オブジェクトはプロパティとメソッドを持ちます。プロパティはオブジェクトの属性を表し、メソッドはオブジェクトの動作を表します。


let person = {
    name: "太郎",
    age: 25,
    greet: function() {
        console.log("こんにちは、" + this.name + "です");
    }
};

console.log(person.name);  // "太郎"を出力
person.greet();  // "こんにちは、太郎です"を出力

このように、オブジェクトを使うことで、データとその操作を一つにまとめることができます。例えば、personオブジェクトは、nameとageというプロパティを持ち、greetというメソッドを持ちます。

オブジェクトのプロパティには、ドット記法を使ってアクセスすることができます。例えば、person.nameとすることで、personオブジェクトのnameプロパティにアクセスできます。

また、オブジェクトのメソッドは、オブジェクトのプロパティにアクセスして操作を行うことができます。例えば、greetメソッドは、this.nameを使ってpersonオブジェクトのnameプロパティにアクセスし、メッセージを出力します。

練習問題

オブジェクトを使って、以下の操作を行ってください。

  1. プロパティtitleとauthorを持つbookオブジェクトを作成してください。
  2. bookオブジェクトに、getSummaryメソッドを追加し、titleとauthorを含む文字列を返すようにしてください。
  3. getSummaryメソッドを呼び出し、結果をコンソールに出力してください。

オブジェクトの作成：
let オブジェクト名 = {
プロパティ名: 値,
メソッド名: function() {
処理
}
};


let book = {
    title: "JavaScript入門",
    author: "山田太郎",
    getSummary: function() {
        return this.title + " by " + this.author;
    }
};

console.log(book.getSummary());  // "JavaScript入門 by 山田太郎"を出力

7. スプレッドシート制御

Google Apps Scriptを使って、Googleスプレッドシートを操作することができます。スプレッドシートのデータを読み書きすることで、さまざまな自動化タスクを実現できます。


// スプレッドシートを開く
let sheet = SpreadsheetApp.openById('スプレッドシートID').getSheetByName('シート名');

// セルの値を取得
let value = sheet.getRange('A1').getValue();
console.log(value);  // セルA1の値を出力

// セルの値を設定
sheet.getRange('A1').setValue('新しい値');

このように、Google Apps Scriptを使うことで、スプレッドシートのセルの値を取得したり設定したりすることができます。例えば、特定のセルの値を読み取って処理を行ったり、計算結果をスプレッドシートに書き込んだりすることができます。

また、スプレッドシートの範囲を指定して、複数のセルの値を一度に操作することもできます。例えば、getRange('A1:B2')とすることで、A1からB2までの範囲を指定できます。

スプレッドシート制御を使うことで、データの管理や分析を効率的に行うことができます。例えば、定期的なレポート作成やデータの集計など、手作業では時間がかかる作業を自動化することができます。

練習問題

スプレッドシート制御を使って、以下の操作を行ってください。

  1. スプレッドシートのセルA1の値を取得し、コンソールに出力してください。
  2. スプレッドシートのセルA1の値を"Hello, World!"に設定してください。

スプレッドシートの操作：
let sheet = SpreadsheetApp.openById('スプレッドシートID').getSheetByName('シート名');
let value = sheet.getRange('A1').getValue();
sheet.getRange('A1').setValue('新しい値');


let sheet = SpreadsheetApp.openById('スプレッドシートID').getSheetByName('シート名');
let value = sheet.getRange('A1').getValue();
console.log(value);  // セルA1の値を出力

sheet.getRange('A1').setValue('Hello, World!');
